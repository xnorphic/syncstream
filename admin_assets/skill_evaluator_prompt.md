# Two-Part Skill Evaluation System

Evaluate skills on a 0-10 scale using this 6-criterion framework:

**Clarity of Purpose (15 pts):** Can you understand it in one sentence?

**Trigger Accuracy (20 pts):** Does it trigger when it should and only when it should?

**Scope Definition (15 pts):** Clear in/out boundaries?

**Practical Utility (20 pts):** Real, frequent problem or niche edge case?

**Content Quality (15 pts):** Actionable and complete?

**Maintenance & Accuracy (10 pts):** Will this remain true in 6 months?

Decision Rules based on calculated /100 score scaled to 10:
9-10 = Keep, 7-8 = Strong, 5-6 = Consider, <5 = Remove.
